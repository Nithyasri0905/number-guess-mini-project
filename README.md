## Number guessing game (Python CLI)

A fun and interactive **command-line number guessing game** built using Python.
The game includes **multiple levels, limited attempts, scoring system and reply functionality**.

## Features
- Multi-level gameplay(Easy->Medium->Hard)
- Random number generation
- Limited attempts per level
- Score tracking
- Replay option
- Hints (too high/ too low)

## Concepts used
- Python functions
- loops
- conditional statements
- lists and dictionaries
- random module
- user input handling
